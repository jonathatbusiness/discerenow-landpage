# DiscereNow Landing Page

Official open-source landing page for the DiscereNow ecosystem.

DiscereNow is a Word-to-SCORM learning content pipeline designed to help instructional designers, learning experience designers, and corporate training teams structure learning content in Microsoft Word and transform it into SCORM-ready or web-based learning experiences through DiscereNow Studio.

---

## About the project

This website introduces the DiscereNow ecosystem, including:

- DiscereNow Template Builder — a Microsoft Word add-in for structured learning content authoring  
- DiscereNow Studio — a beta tool for transforming structured content into web and SCORM packages  
- Privacy information required for marketplace publication  
- Download areas for future public releases

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
